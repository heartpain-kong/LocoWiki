# 🛠️ Supporting Scripts (scripts/)

> [中文](README.md) | English

This folder contains helper scripts for maintaining this repository.

## fetch_open_access_pdfs.py

Fetches publicly available paper PDFs based on the unified paper list in `reading-list/papers.json` (e.g., arXiv / OpenReview when available).

- Show help: `python3 scripts/fetch_open_access_pdfs.py --help`
- Download PDFs: `python3 scripts/fetch_open_access_pdfs.py --download`
